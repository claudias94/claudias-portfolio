# Scope & Authorization

## Purpose

This repository is a **personal cybersecurity portfolio** created for
educational, demonstration, and professional showcase purposes.

All security testing activities referenced here were conducted in **authorized,
controlled, and ethical environments**.

---

## Authorized Scope

Security testing described or demonstrated in this repository was performed only on:

- Personal lab environments
- Intentionally vulnerable platforms (e.g., Hack The Box, training labs)
- Self-developed applications
- Systems where **explicit permission** was granted

No testing was performed on live production systems without authorization.

---

## Prohibited Activities

The following activities are **strictly prohibited** and not endorsed:

- Unauthorized penetration testing
- Attacking systems without written permission
- Scanning or exploiting third-party infrastructure
- Abuse of vulnerabilities for malicious purposes

---

## Legal & Ethical Compliance

All work aligns with:
- Ethical hacking principles
- Responsible disclosure standards
- Applicable local and international cybersecurity laws

This repository does **not** encourage illegal activity.

The author assumes **no responsibility** for misuse of the information,
techniques, or code contained in this repository.

Any actions taken based on this material are done **at the user's own risk**.
